AdTech Competitor Analysis & Lead Generation
Overview
This project provides an automated solution for identifying and qualifying potential publisher leads within the AdTech ecosystem. By analyzing competitor runtime domains and integrating with internal APIs, the script streamlines the process of market research and contact discovery.

Key Features
Competitor Domain Analysis: Identifies sites working with specific competitors by fetching data through API integrations.

Client Filtering Logic: Automatically cross-references discovered leads against an existing client database to ensure sales efficiency.

Automated Lead Enrichment (OSINT): Scrapes publisher pages to extract critical contact information, including emails and social media profiles.

Data Processing: Handles large-scale analysis (integrated into a 460+ line Python framework).

Technical Stack
Language: Python 3.x

Key Capabilities: API Integration, Web Scraping, Data Filtering, and Contact Extraction.

How It Works
Fetch: The script retrieves a list of domains associated with competitors.

Validate: It filters out domains that are already active clients.

Extract: It performs a deep scan of the target websites to find relevant business contacts.
