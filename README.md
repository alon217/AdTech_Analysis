# AdTech Competitor Analysis & Lead Generation

## Overview
This project provides an automated solution for identifying and qualifying potential publisher leads within the AdTech ecosystem.

## Key Features
* **Competitor Domain Analysis**: Identifies sites working with specific competitors via API.
* **Client Filtering Logic**: Cross-references discovered leads against existing client databases.
* **Automated Lead Enrichment**: Scrapes publisher pages to extract emails and social links.

## Technical Stack
* **Language**: Python 3.x
* **Libraries**: Requests, BeautifulSoup4, Pandas
